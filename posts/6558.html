class set([iterable])
集合对象。

>>> s = {1,2,3}
>>> type(s)
<\class ‘set’>

setattr(object, name, value)
设置对象属性值，name参数必须为对象存在属性的字符串格式，value可谓

等价于:

>>> import math
>>> math.pi
3.141592653589793
>>> math.pi = 3.14
>>> math.pi
3.14

class slice(start, stop[, step])
切片函数。返回切片对象，类似于range(start, stop, step)。

等价于

>>> l[3:9:2]
[7, 11, 15]

*sorted(iterable, , key=None, reverse=False)
排序函数。返回可迭代对象的元素排序后的新列表。

>>> r = [1, 8 ,134, 33, 456, 4]
>>> sorted(r)
[1, 4, 8, 33, 134, 456]

super([type[, object-or-type]])
super() 函数是用于调用父类或同级类的一个方法（代理对象）。
① 直接引用：
函数super()相当于一个桥梁，连接需要调用的函数后执行。

>>> b = B()
>>> b.add_b(2)
3

② 单类继承：
函数super()取继承的子类值，执行其继承函数。

>>> b = B()
>>> b.add(4)
>>> print(b.n)
10

③ 多类继承：
函数super()使用多继承，会涉及到查找顺序（MRO）问题。

>>> d = D()
>>> d.add(5)
>>> print(d.n)
22

查找顺序:

>>> print(D.mro())
>>> [< class ‘__main__.D’>, < class ‘__main__.B’>, < class ‘__main__.C’>, < class ‘__main__.A’>, < class ‘object’>]
