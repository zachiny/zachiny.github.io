abs(X)
返回一个数值的绝对值。

>>>abs(-10)
10

**all( iterable )**
若 iterable 的 所有 元素为真（或可迭代对象可为空），则返回True。

any(iterable)
若 iterable 的 任一 元素为真，则返回True；且___iterable___为空，返回False。

ascii(object)
鸡肋函数
返回一个对象的可打印表达形式的字符串， 由于 ASCII 编码字数少，故不支持的编码返回使用\x, \u 或 \U转义,与函数repr() 类似。

>>>ascii(‘object’)
“‘object’”
>>>ascii(‘对象’)
“‘\u5bf9\u8c61’”

bin(x)
返回一个将整数转换为带‘0b’的二进制的字符串。

>>>bin(233)
‘0b11101001’

去“0b”前缀：
>>>format(14, ‘#b’), format(14, ‘b’)
(‘0b1110’, ‘1110’)
>>>f’{14:#b}’, f’{14:b}’
(‘0b1110’, ‘1110’)

class bool([x])
返回对象的布尔值，True 或 False。
若x为假或空，则为 False。

>>>bool(233)
True
>>>bool(‘’) # 空字符串
False

breakpoint(*args, **kws)
函数执行调用中断。
应用意义不明

class bytearray([source[, encoding[, errors]]])
返回字节数组，可变序列。
应用意义不明

class bytes([source[, encoding[, errors]]])
返回字节对象，等价于bytearray，不同点是不可变序列。
应用意义不明

callable(object)
如果对象可被调用，返回 True。
应用意义不明

chr(i)
返回由整数i，使用 Unicode 编码的特征码字符串。

>>>chr(54321)
‘퐱’

@classmethod
转换一个模块为类模块。

compile(source, filename, mode, flags=0, dont_inherit=False, optimize=-1)
编译代码。

class complex([real[, imag]])
返回复数。

>>>complex(1, -1)
(1-1j)
