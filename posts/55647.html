class memoryview(obj)
返回包括字节或或字节组的内部数据(类似于编码)结果。

*min(arg1, arg2, args、[, key])
返回参数最小值，若参数为迭代器，则返回最大的迭代器。

next(iterator[, default])
返回从iterator的_next_() 模块检索的下一个项目。

class object
返回一个无特征对象。
鸡肋函数

>>>object()
<\object object at 0x000000000048A100>

oct(x)
返回一个将整数转换为带‘0o’的八进制的字符串。

>>>oct(233)
‘0o351’

open(file, mode=’r’, buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)
打开文件返回文件操作对象。
open为文件操作起点。
close为文件操作终点。

mode——打开文件模式：

模式 描述
‘r’ 打开读取（默认），不存在文件报错
‘w’ 打开写入，从文件首行覆盖写入
‘x’ 创建打开，存在文件则报错
‘a’ 追加写入，存在文件从尾行写入
‘b’ 二进制模式
‘t’ 文本模式（默认）
‘+’ 打开更新（即读和写）

buffering——缓冲设置，不写默认即可。
encoding——编码格式，一般使用utf8。
errors——报错级别，不写默认即可。
newline——换行控制，默认换行符’\n’。
closefd——文件描述符号，不写默认即可。
opener——设置自定义开启器。

seek()函数是指针函数，指向读取位置。

ord(c)
返回使用 Unicode 编码的特征码字符串c的整数。

>>>ord(‘⑨’)
9320

pow(base, exp[, mod])
次方函数。返回参数base的exp次方结果，若含第三个参数，基于结果再取模值。

*print(objects, sep=’ ‘, end=’\n’, file=sys.stdout, flush=False)

打印函数。没啥可讲
默认换行打印end=’\n’，可重写为end=’\t’实现同行花里胡哨打印。

class property(fget=None, fset=None, fdel=None, doc=None)

返回属性值。
说明：https://www.runoob.com/python/python-func-property.html

class range(start, stop[, step])
range 实际上是不可变序列类型。
start：开始（开区间）
stop：结束（闭区间）
step：间隔

repr(object)
返回一个对象的可打印表达形式的字符串，显示打印范围广，可在自定义类中重写。

>>>repr(‘object’)
“‘object’”
>>>repr(‘对象’)
“‘对象’”

reversed(seq)
返回一个反转的 iterator，参数 seq 必须是一个有 __reversed__() 的 对象 或支持序列协议（具有从 0 开始的整数类型参数的 __len__() 方法和 __getitem__() 方法）。

round(number[, ndigits])
截位函数。返回截位结果。

>>> round(6.66, 1)
6.7
>>> round(-6.66, 1)
-6.7
