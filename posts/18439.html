Python面向对象

面向对象介绍：

面向对象是一种编程思想，根据现实中人类对事物的理解，基于逻辑思维方式抽象化计算机编程语言，以此构建软件系统。

比如说我们需要用编程语言描述一类事物——车。此处的“车”仅仅只是一个抽象概念，因为你不知道车是两轮的人力自行车，三轮的电瓶三轮车，还是四轮的电动汽车。

面向对象是将抽象的事物加以具现化的技术方法。

定义类：

在面向对象方法中，类是描述对象的集合。
接下来简单地来定义一个类：

这是一个没有进行操作的空白类。

>>> print(Car)
<\class ‘__main__.Car’>
>>> print(Car())
<__main__.Car object at 0x0000000001227C50>

创建对象：

对象是类的实例。

接下来简单地来创建一个类的对象：

>>> tesla = Car()
>>> print(tesla)
<__main__.Car object at 0x0000000001227FD0>

成员变量：类变量和实例变量组合的统称。
成员方法：类方法和实例方法组合的统称。
属性：特指property，而非attribute（特征，特值）。

注：其实类的所谓“方法”其实就是函数。

实例变量

实例变量：实例特有的变量。

tesla是一个实例，实例变量是对象个体特有的“数据”，即上面实例运行类方法，即info()函数时的tesla.country, tesla.brand。

构造方法

构造方法：类的__init__(self)方法（函数）。

Python的__init__()方法是一个非常特殊的方法，类实例化时会自动调用，与4.1 实例变量例子不同，byd实例只需要直接声明就可以调用，而不需要2步。

实例方法

实例方法：实例特有的方法（函数）。

类变量

类变量：类定义的公有变量，属于类，不属于个体实例。

注：像实例变量、类变量，与实例方法，类方法之间的区别用一个简单的例子来说明。
比如在一个定义了类变量，轮子 wheel = 4 的汽车类，有这个类的2个实例特斯拉tesla，比亚迪byd。2个实例虽然都有名称，数值完全一样的实例变量，但是就像我们不会说特斯拉汽车轮子等于比亚迪的汽车轮子一样，tesla这个实例不等于byd这个实例。
类方法和实例方法区别同理。

类方法

类方法：即类的公有函数。

使用classmethod修饰符的方法（函数）是类方法，同类变量一样，不需要实例化；且类方法不需要 self 参数，但第一个参数需要是表示自身类的 cls 参数，可以来调用类的属性，类的方法，实例化对象等。

封装性：

Python面向对象三大特征：封装，继承，多态。
封装性是指类的变量，方法（函数），属性只能使用类或实例调用，不允许类外或实例外调用。
下面来浅析Python封装性。

私有变量

私有变量，变量名称添加“__”2个下划线。

若引用私有变量则引发错误：

>>> print(Optimus_Prime.__engine)
AttributeError: ‘Car’ object has no attribute ‘__engine’

>>> print(Car.__engine)
AttributeError: type object ‘Car’ has no attribute ‘__engine’

私有方法

类似于私有变量，方法（函数）前添加“__”2个下划线。

私有方法直接调用会引发错误：

>>> print(Optimus_Prime.__desc())
AttributeError: ‘Car’ object has no attribute ‘__desc’
>>> print(Car.__desc())
AttributeError: type object ‘Car’ has no attribute ‘__desc’

使用属性

使用私有变量，私有方法是为了实现对象的封装，那么封装后如何访问？

方法一：通过公有的set（赋值）和get（取值）方法访问。

方法二：使用属性(更好，更方便)。

继承性：

接下来浅析Python面向对象的继承性。

举个例子：
比亚迪汽车（类）是一种品牌汽车（类），具有汽车（类）的全部特点，即属性和方法。
汽车都有4个轮子，有引擎，有刹车等，比亚迪特斯拉汽车当然都有。
在面向对象中汽车是一般类，称之为“父类”；比亚迪汽车是特殊类，称之为“子类”。
特殊类拥有一般类的全部属性和方法，故曰：子类继承父类，即继承性。

继承

多继承

多继承，即多个继承。
举个例子：
混合动力汽车是一种汽车，这种汽车既使用汽油驱动又使用电力驱动，既继承了汽油驱动车的特点，有继承了电力驱动车的特点。
多继承由于继承多个对象，因此继承存在优先性。

方法重写

多继承对父类方法（函数）进行重写，即修改方法内容。

多态性：

最后浅析Python面向对象的多态性。

举个例子：
比亚迪汽车，特斯拉汽车，丰田汽车都是汽车，这三种汽车都有品牌，价格，种类，但是它们“品牌”，“价格”，“种类”各不相同。
子类出于父类而别于父类，故称之为：对象表现出多种形态，即多态性。

继承与多态

在多个子类继承父类，并重写父类方法后，这些子类所创建的对象之间就是多态的。这些对象采用不同的方式实现父类方法。

鸭子类测试与多态

鸭子类型测试：若看到一只鸟走起来像鸭子、游泳起来像鸭子、叫起来也像鸭子，那么这只鸟可以被称为鸭子。
Python支持鸭子类型测试，Python解释器不检查发生多态的对象是否继承了同一个父类，只要它们有相同的方法（函数）
它们之间就是多态的。
