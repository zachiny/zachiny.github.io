hash(object)
返回对象经过散列算法后变换的散列值，又译哈希值。

>>>hash(‘哈希算法’)
2428179105798671580

help([object])
返回对象信息介绍。

hex(x)
返回整数x转换为16进制结果。

>>>hex(233)
‘0xe9’

去结果0x，或转大小写方法。
>>> ‘%#x’ % 255, ‘%x’ % 255, ‘%X’ % 255
(‘0xff’, ‘ff’, ‘FF’)
>>> format(255, ‘#x’), format(255, ‘x’), format(255, ‘X’)
(‘0xff’, ‘ff’, ‘FF’)
>>> f’{255:#x}’, f’{255:x}’, f’{255:X}’
(‘0xff’, ‘ff’, ‘FF’)

id(object)
返回对象的标识信息。

>>>id(233)
18435576

input()
可携带提示输入后，返回输出。

class int([x])
返回整数。

>>> int(233)
233
>>> int(-233)
-233

isinstance(object, classinfo)
若object参数是classinfo的实例，返回True，否则False。
classinfo可为元组，满足任一返回True。

issubclass(class, classinfo)
class参数是classinfo的子类，返回True，否则False。

>>> from collections import Counter
>>> issubclass(Counter, dict)
True

iter(object[, sentinel])
返回一个 iterator 对象。

len(s)
返回对象长度。

>>>len([1,2,3])
3

class list([iterable])
列表对象。

>>> l = [1,2,3]
>>> type(l)
<\class ‘list’>

locals()
刷新当前环境下所有类和参数字典。于globals()环境一致。

>>> locals()
{‘doc‘: None, ‘builtins‘: <module ‘builtins’ (built-in)>, ‘loader‘: <class ‘_frozen_importlib.BuiltinImporter’>, ‘name‘: ‘main‘, ‘package‘: None, ‘spec‘: None}

map(function, iterable, …)
映射函数。返回对应 iterable 的每个对象执行参数 function 后的函数。

*max(arg1, arg2, args、[, key])

返回参数最大值，若参数为迭代器，则返回最大的迭代器。
